# Mytty を使い始める

English version: [getting-started.md](getting-started.md)

このチュートリアルでは Mytty をインストールし、初めて起動し、window を tab と
pane に分割するところまでを手を動かして進めます。最後には、キーボードだけで
pane を移動し、拡大や入れ替えを試し、一度終了して再起動しても元の状態に戻る
ことを確認できているはずです。

## インストールする

Mytty には macOS 15 以降・Apple Silicon が必要です。
[Releases](https://github.com/m-tkg/Mytty/releases) から `Mytty.zip` を
ダウンロードして展開し、`Mytty.app` を `/Applications` にドラッグしてください。
署名と notarize 済みのビルドなので、ダブルクリックすればそのまま開き、
Gatekeeper の警告をクリックして回避する必要はありません。ソースからビルド
したい場合は [macOS アプリをソースからビルドする](../how-to/build-macos-app_ja.md)を先に済ませてから戻って
きてください。

## 起動する

`/Applications` または Spotlight から Mytty を開きます。事前の設定は何も
要りません。最初に開くウィンドウには、すでにログインシェルが起動していて
すぐ入力を受け付ける状態になっています。今日はここまでで終えても、ひとまず
使える terminal が手元にあることになります。

## window を tab と pane に分割する

Command-T で 2 つ目の tab を開きます。新しい tab はウィンドウ上部の帯では
なく、左側のサイドバーに行として増えます。これは意図的な設計で、長い
タイトルでも読めますし、数が増えても一覧の位置が崩れません。各行にはその
tab が持つ pane 数も表示されます。

続けてその tab の中で Command-D を押します。フォーカスしている pane が
左右に半分ずつ分割され、2 つの pane が並びます。

![サイドバーに 2 つのタブ、pane が左右に分割された画面](../images/panes.png)

分割した tab の上で、次を順番に試してみてください。

| やってみること | ショートカット |
| --- | --- |
| 上下に分割する | Command-Shift-D |
| 上下左右の pane にフォーカスを移す | Command-Option-矢印 |
| フォーカス中の pane を全体表示、もう一度で解除 | Control-Command-Return |
| すべての pane のサイズを均等にする | Control-Command-= |
| フォーカス中の pane を閉じる | Command-Shift-W |

pane を閉じたら Command-Shift-T を押してみます。直前に閉じた pane が、
スクロールバックも分割位置も保ったまま復元されます。同じ shortcut は tab を
閉じたときにも使え、複数閉じてしまった場合は Window メニューの
**最近閉じた項目**サブメニューから直近 20 件までを個別に選んで戻せます。

それまで何をしていたとしても一度試す価値があるのが Command-Shift-P です。
すべてのメニューコマンドを一覧できるコマンドパレットが開き、入力した文字で
絞り込めます。おおよその操作内容が分かっていればメニューバーをたどるより
速く、名前や shortcut を忘れたコマンドを探す手段としても実用的です。

![pane の上に開いたコマンドパレット。各メニューコマンドと shortcut の一覧](../images/command-palette.png)

pane が増えてどれがどれか分からなくなったら、Control-Command-P で
**すべてのペインを表示**を開きます。各 pane の現在の command または Agent 名
と作業ディレクトリが一覧になり、行を選ぶとその tab へ移動して pane に
フォーカスします。

pane が 2 つ以上並んだ状態で Control-Command-S を押すと**ペインを入れ替え**
モードに入ります。最初にクリックした pane にアクセントカラーの枠が付き、
ステータスヒントが 2 つ目の選択を促します。2 つ目の pane をクリックすると、
分割比率を保ったまま 2 つの位置が入れ替わります。矢印キーと Return でも同じ
操作ができるので、マウスなしでも完結します。同じ pane をもう一度クリック
するか、コマンドを再度実行するとキャンセルできます。

ウィンドウ下端のバーはフォーカス中の pane に追従し、その作業ディレクトリと、
Git 管理下の場合はリポジトリと branch を表示します。

## 終了して再起動する

このチュートリアルで開いた tab と pane を残したまま Command-Q で Mytty を
終了し、もう一度起動してみてください。window、tab、pane、それぞれの分割比率
と作業ディレクトリはすべて元通りに復元されます。pane で Agent のセッションが
動いていた場合も、resume 可能な状態で戻ってきます。アプリを終了することを
身構える必要はありません。
