# Mytty を使い始める

English version: [getting-started.md](getting-started.md)

## インストールする

Mytty には macOS 15 以降・Apple Silicon が必要です。[Releases](https://github.com/m-tkg/Mytty/releases) から最新版の `Mytty.zip` をダウンロードして展開し、`Mytty.app` を `/Applications` にドラッグしてください。

ソースからビルドしたい場合は [macOS アプリをソースからビルドする](../how-to/build-macos-app_ja.md)を参照してください。

## 起動する

Mytty のアプリをダブルクリックして起動します。

cmd+shift+P でコマンドパレットが起動します。ここから様々なコマンドを実行できます。キーボードショートカットや、メニューバーからも操作を行えます。

## ウィンドウ、タブ、ペイン

ターミナルが表示されている部分をペインと呼びます。ウィンドウは複数のタブを持ち、各タブは複数のペインに分割できます。

「新規ウィンドウ」で新規のウィンドウを開きます。

「新規タブ」で新規のタブを開きます。

「右に分割」で現在アクティブなペインの右に新規でペインを作成します。

タブ内のペインは、上下左右に複数作成できます。

ペインの移動や入れ替えなどもキーボードショートカットで行えます。設定からどんなキーボードショートカットがあるのか確認して色々試してみてください。

![サイドバーに 2 つのタブ、ペインが左右に分割された画面](../images/panes.png)

タブやペインを閉じた後、「最近閉じた項目」から復元できます。

この機能が有効なのは Mytty が起動している間だけです。Mytty のアプリを終了するとこの履歴は削除されます。


![ペインの上に開いたコマンドパレット。各メニューコマンドとショートカットの一覧](../images/command-palette.png)

タブやペインが増えた場合、「すべてのペインを表示」を開くことでペインの一覧を表示して、そこに移動できます。


## ステータスバー

ステータスバーはフォーカス中のペインに追従し、その作業ディレクトリと、Git 管理下の場合はリポジトリとブランチを表示します。

GitHub のアイコンをクリックすると GitHub のリポジトリページが開きます。

フォルダのアイコンをクリックすると Finder でそのディレクトリを開きます。

AI Agent を起動している場合、利用中の Agent 名やモデル、コンテキストやリミットなどが表示されます。

月のアイコンをクリックすると、AI Agent 起動中にスリープを防ぐかどうかの設定ができます。

## 終了して再起動する

タブやペインを残したまま Mytty を終了し、もう一度起動すると、タブの状態やペインの配置、AI Agent のセッションも含めて復元されます。
