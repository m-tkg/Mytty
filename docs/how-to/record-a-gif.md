# Record a pane as a GIF

This covers capturing whatever is happening in a pane, whether that's a
build, a demo, or an agent working, as a shareable GIF, plus the setting
that overlays which keys were pressed.

## Start and stop a recording

Focus the pane you want to capture and use **Start/Stop Recording** from the
menu. A stop control appears on the tab while recording is active, so it's
obvious a capture is running even after you've moved on to something else.

Recording stops automatically after 60 seconds if you don't stop it first,
and either way a save panel appears right after, asking where to write the
GIF file.

## Show pressed keys in the recording

Turn on **Show pressed keys in pane** in Settings to overlay key labels below
the cursor as you type, during normal use and not just while recording.
Whatever appears on screen this way is captured at the same position in the
GIF, which is what makes a recorded demo readable without a voiceover
explaining what was typed.
