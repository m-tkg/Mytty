# pane への入力を日時指定で送る

status bar の時計メニューから、pane への文字送信を指定した日時に予約できる。
会議が終わった後に打ちたい command や、その場を離れている間にエージェント
へ送りたい続きの指示に使える。

## 予約を作る

status bar の時計メニューを開き、日時、送信する文字列、末尾に改行を付ける
かどうかを入力する。改行を付けると、文字列を打った上で Return を押したのと
同じ挙動になる。予約の対象になるのは、作成した時点でアクティブだった pane
である。

## 予約を管理する

登録済みの予約は同じ時計メニューに並び、そこから編集・削除できる。知って
おくとよい挙動がいくつかある。

- 過去の予約や送信済みの予約は自動的に消えるので、メニューには常に未実行
  の分だけが残る。
- 予約の対象になっている pane を閉じると、その予約も一緒に削除される。
  存在しない pane に対して送信が実行されることはない。
- 指定した時刻に Mytty が起動していなければ何も送信されない。次回の起動時
  にまとめて遅れて実行されるわけではない。
