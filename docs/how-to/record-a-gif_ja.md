# ペインを GIF として録画する

ビルドやデモ、動いているエージェントなどペイン内の様子を、GIF アニメーションとして残せます。

![Mytty 自身で録画した GIF。ペインに 2 つのコマンドを入力する様子と、押したキーがカーソルの下に表示されている](../images/record-a-gif.gif)

## 録画を開始・停止する

録画したいペインをフォーカスし、「録画を開始/停止」を実行します。録画中はタブに停止ボタンが出ます。

![録画中のサイドバーのタブ。赤い停止ボタンが表示されている](../images/recording-stop-button.png)

自分で止めるか 60 秒経過で自動的に録画が停止します。停止後に保存パネルが開き、GIF ファイルの保存先を入力します。

## 押したキーを録画に表示する

設定の「押したキーをカーソルの下に表示」を有効にすると、入力するたびにカーソルの下に入力したキー名が表示されます。これは録画中に限らず通常時でも利用可能ですが、録画やデモなどで便利です。
