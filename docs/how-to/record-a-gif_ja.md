# pane を GIF として録画する

build やデモ、動いているエージェントなど pane 内の様子を、共有しやすい GIF
として残す手順と、押したキーを画面に重ねて表示する設定を扱う。

## 録画を開始・停止する

録画したい pane をフォーカスし、メニューの **録画を開始/停止** を使う。
録画中は tab に停止ボタンが出るので、別の作業に気を取られていても録画中
であることが分かる。

自分で止めなければ 60 秒で自動的に停止する。どちらの場合も停止直後に保存
パネルが開き、GIF ファイルの保存先を尋ねられる。

## 押したキーを録画に表示する

設定の **押されたキーを pane に表示** を有効にすると、入力するたびに
cursor の下にキー名が重なって表示される。これは録画中に限らず通常の操作
でも同じで、画面に出ているものがそのままの位置で GIF にも含まれる。デモを
録画したとき、何を打ったかを別途説明しなくても再生だけで読み取れるのは
この設定のおかげである。
